## 常用技巧

### 换行

- 连续两个以上空格+回车
- 使用br标签

### 缩进字符

```
&#160;和&nbsp; 缩进1/4个中文，1/2个英文
&#8194;和&ensp; 缩进1/2个中文，1个英文
&#8195;和&emsp; 缩进1个中文，2个英文
```

例如
```
你好你好<br>
你好&nbsp;你好<br>
你好&ensp;你好<br>
你好&emsp;你好
```

渲染后如下

你好你好<br>
你好&nbsp;你好<br>
你好&ensp;你好<br>
你好&emsp;你好
