正则小册子

发表于2020-11-26
|字数总计:708|阅读量:

正则可以说是一个比较通用的规则，用于匹配指定的内容

基本使用

介绍个网站模拟练习 regex101

a
a A
逐个字符匹配，区别大小写

元字符

特殊含义的字符

元字符 含义
. 匹配单个字符(除了换行符)
[] 匹配中括号内的任意字符
[^] []取反
* 任意个字符
+ 至少一个字符
? 0或1个字符
{n,m} 匹配x个字符或字符集(n<=x<=m)
(xyz) 匹配字符集
&#124; 或
|转义
^ 从开始行匹配
$ 从末尾开始匹配

点运算符.

.ava
Java java

在[ ]中表示英文句号
ava[.]
Java java.

字符集[]

[Jj]ava
Java java

否定字符集[^]

[^J]ava
Java java

重复次数*

[a-z]*
Java java

重复次数+

v.+a
Java java

重复次数?

[J]?ava
Java java

{}一个或一组字符出现的次数

[a-z]{2,3}
JAva java

[a-z]{2,}
JAva java

[a-z]{3}
JAva java

()群组

(J|j)+
Java java

(av){2}
Javava

|或运算

(J|j)ava|gava
Java java gava

\转义

(J|j)ava.?
这个和(J|j)ava[.]?作用一样
Java java.

^和$

^(J|j)ava
Java java

(J|j)ava$
Java java

字符集

字符集 含义
\w 数字字母下划线[a-zA-Z0-9_]
\W \w取反[^\w]
\d 数字[0-9]
\D \d取反[^\d]
\s 所有空格字符[\t\v\n\r\f\p{Z}]
\S \s取反[^\s]
\f 换页符
\n 换行符
\r 回车符
\t 制表符
\v 垂直制表符
\p 匹配 CR/LF（等同于 \r\n ）,⽤来匹配 DOS ⾏终⽌符

\p{P}
小写 p 是 property 的意思,表示 Unicode 属性,用于 Unicode 正表达式的前缀,中括号内的“P”表示Unicode 字符集七个字符属性之一：标点字符,注：此语法部分语言不支持,例：javascript
其他六个属性：

  • L：字母
  • M：标记符号（一般不会单独出现）
  • Z：分隔符（比如空格、换行等）
  • S：符号（比如数学符号、货币符号等）
  • N：数字（比如阿拉伯数字、罗马数字等）
  • C：其他字符

零宽度断言

断言 含义
?= 正先行断言
?! 负先行断言
?<= 正后发断言
?<! 负后发断言

断言可以理解为一种对正则匹配的限制,断言本身不会匹配到结果中

(J|j)ava(?=\sjava)
Java java

(J|j)ava(?!\sjava)
Java java

(?<=Java\s)(J|j)ava
Java java

(?<!Java\s)(J|j)ava
Java java

标志

别名模式修正符，上述所有例子是基于gm模式的

标志 含义
i 忽略大小写
g 匹配全部可匹配结果
m 多行匹配

/Java/ig
Java java

/.ava/gm
Java
java

贪婪与惰性

正则默认是贪婪匹配，贪婪的原子性是单行的

贪婪
/.*va/
Java java.

/.*va/
Java
java.

惰性
/.*?va/
Java java.

/.*?va/g
这个会先匹配”Java”和” java”(.是会匹配中间的空格的)
