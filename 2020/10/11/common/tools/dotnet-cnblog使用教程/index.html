dotnet-cnblog使用教程

发表于2020-10-11
|字数总计:300|阅读量:

安装.Net Core SDK

下载地址： https://dotnet.microsoft.com/learn/dotnet/hello-world-tutorial/install

安装完后确认版本号

dotnet –info

安装dotnet-cnblog

open you cmd

dotnet tool install -g dotnet-cnblog

配置博客园相关信息

先随便写一个带图的markdown文件，在cmd里cd到md文件所对应的文件夹，然后操作命令

dotnet-cnblog test.md

然后你就发现你的cmd自动变黄了，没问题不要慌

然后他就让你输id(你博客主页地址域名最后一个斜杆后面的东东)，还有什么用户名密码如此如此的，用的tea算法加密的，安全的很

Reset

这个就easy了(如果你感脚信息错了)

dotnet-cnblog reset

生成的文件

采用的是test-cnblog.md的格式在当前目录下生成的一个新的文件，里面的本地图片地址都已经替换为https开头的能在网址上访问的图片了，大工告成，尽情ctrl+c再ctrl+v吧，你也会成为博客园的大水军

标准结尾

比心提示哦，用markdown的小伙伴推荐下载typora软件(真香)，在对本地图片存储上建议如下
